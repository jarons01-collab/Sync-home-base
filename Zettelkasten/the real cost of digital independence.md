- For someone who is already used to shaving using a cartridge razor, the learning curve for using a safety razor isn't that steep.
- Not so with GitHub. 
	- By design, every change to the contents of a repository *must* include a commit memo.
- The [Obsidian team](https://obsidian.md/about#team) has done an exemplary job of making Obsidian's learning curve not as steep, but tailoring it to one's own needs is very much an individualized process.
- Building your own server rack also has a steep learning curve, so it's much easier to lease part of one that's already pre-assembled.
