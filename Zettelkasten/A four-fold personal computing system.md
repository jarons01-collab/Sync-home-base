Inspired by the new [Steam Hardware](https://store.steampowered.com/hardware), I have envisioned a FOSS-based computing ecosystem which includes the following components:
- A home server (the room)
- A rendering engine (a custom-built gaming PC running NixOS)
- A handheld ultra-mobile PC similar to a [Samsung Q1 ultra](https://en.wikipedia.org/wiki/Samsung_Q1)
	- With an accompanying docking station
- A smartwatch running WebOS with three interchangeable PCBs:
	1. The Display module
	2. The Processor module
	3. The Biometrics module
